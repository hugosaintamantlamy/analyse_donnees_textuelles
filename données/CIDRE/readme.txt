The Corpus for Idiolectal Research (CIDRE)

Dans ce corpus, il y a plus de 400 œuvres de fiction de 11 auteurs prolifiques du 19e et du début du 20e siècle. Nous avons daté ces œuvres par leur date d’écriture. Cela permet d’étudier l’évolution de l’idiolecte des différents auteurs.

Contributeurs : Olga Seminck, Philippe Gambette, Dominique Legallois, Thierry Poibeau.

Métadonnées disponibles dans le fichier metadata.csv :
- Titre : titre de l'ouvrage contenu dans le fichier
- Auteur : auteur ou autrice selon la dénomination sur data.bnf.fr
- URL : URL de l'édition source
- Nom fichier : nom du fichier
- URL source année fin écriture : URL de la source utilisée pour dater l'œuvre
- Année fin écriture : année de fin de l'écriture selon la meilleure source trouvée, à défaut année de première publication de l'œuvre
- Nom auteur dans corpus : nom de famille de l'auteur ou autrice
- Id auteur wikidata : identifiant de l'auteur ou autrice (données du 12/04/2021)
- URI œuvre data.bnf : url identifiant l'œuvre dans la base data.bnf.fr, si elle y est référencée (données du 12/04/2021)
- Nb words : nombre de mots dans le fichier selon le comptage par la tokenisation de spacy  (https://spacy.io) en excluant la ponctuation. 
- Nb tokens : nombre de tokens dans le fichier selon la commande Unix wc -w
- Licence : domaine public
